## Adding Images Later

### For research figures:
Replace placeholder in HTML with:
```html
<img src="your-figure-name.jpg" alt="Description">
```

### For conference photos:
Replace placeholder in HTML with:
```html
<img src="conference-photo.jpg" alt="Conference Name">
```

Upload images to the repository alongside index.html.
